# SmallBets Portfolio

**170+ projects. One philosophy: Ship fast, learn faster.**

This is Victor del Rosal's public portfolio showcasing the SmallBets approach to building - making many small bets instead of one big one.

## Categories

- **AI & LLM** - Multi-agent systems, memory frameworks, chatbots
- **Productivity** - Notes, task managers, time tracking
- **Education** - Learning platforms, courses, academies
- **Games** - Arcade classics, puzzles, interactive experiences
- **Health** - Fitness tracking, habit building, wellness
- **Business** - SaaS tools, marketplaces, automation
- **Dev Tools** - Utilities, extensions, monitoring
- **Web Apps** - Landing pages, portfolios, interactive sites

## The Philosophy

Instead of one big bet, make many small ones. Each project teaches something. Some fail. Some succeed. All compound into expertise.
